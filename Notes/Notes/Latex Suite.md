| --------- | ---------------- |

### Brackets and Delimiters

| Trigger | Replacement           |
| ------- | --------------------- |
| avg     | \langle $0 \rangle    |
| norm    | \lvert $0 \rvert      |
| mod     | \| $0 \|              |
| Norm    | \lVert $0 \rVert      |
| set     | \{ $0 \}              |
| floor   | \lfloor $0 \rfloor    |

| lr{     | \left\\{ $0 \right\\} |

Press any of the following whilst having text highlighted to enclose the highlighted text in the brackets:
- (
- {
- \[

### Greek Letters

All greek letters can be typed by just typing the letter out.
- “alpha” -> ”\alpha”
Doesn’t work for some that contain other symbols/letters in their name.
- “epsilon” -> “e/psi lon”

Can also type semicolon before the first letter of the english spelling of the letter.
- “;a” -> “\alpha”
Make the letter a capital for capital greek letter.
- “;G” -> “\Gamma”
For ”var” and ”di” letters, use a colon instead.
- “:k” -> “varkappa”
This doesn’t work for all letters
- E.g. psi, phi, pi, eta, omicron
